# brownlu-blog Cheatsheet

## Terminal Commands
- `help` ‚Äî show available commands
- `ls` ‚Äî list blog posts
- `cat <slug>` ‚Äî read a post
- `cd <path>` ‚Äî navigate
- `whoami` ‚Äî about me
- `date` ‚Äî current time
- `tag <name>` ‚Äî browse by tag
- `get <script>` ‚Äî fetch a script
- `activity` / `heatmap` ‚Äî post frequency
- `clear` ‚Äî clear screen
