刚拿到一台新电脑，最让人头疼的莫过于——我原来装了什么软件来着？

与其每次重装系统后凭记忆一个一个去下载，不如趁这次好好整理一份自己的效率工具箱。本文推荐的工具都是我个人反复使用后留下来的，原则是：**免费优先、开源加分、轻量高效、一次配置长期受益**。

## 包管理：Scoop

Windows 最大的痛之一就是没有像样的包管理器。每次重装系统，你要打开浏览器，搜索 Firefox，找下载链接，点安装，下一步下一步……装 30 个软件就得重复 30 次。

Scoop 解决了这个问题。它是一个命令行包管理器，一条命令装一个软件：

```powershell
scoop install firefox 7zip vscode everything sumatrapdf
```

把常用软件写到一个 `scoopfile` 里，新电脑上跑一遍就全装好了。更妙的是 Scoop **安装的软件都在用户目录下**，不需要管理员权限，不会污染系统注册表。

### 安装 Scoop

Windows 下推荐使用 Gitee 镜像源安装：

```powershell
irm https://gitee.com/glsnames/scoop-installer/raw/master/bin/install.ps1 | iex
```

### 国内加速配置

由于 Scoop 默认从 GitHub 拉取 bucket 和下载软件，国内网络经常超时。安装后的第一件事就是配置加速。

**切换到 xrgzs/scoop 优化版（推荐）**：这是一个特供优化版，添加了 hook **自动判断并替换**下载链接为国内源，基于 ISP 检测加速，无需手动修改 bucket。

```powershell
# 替换 scoop 源码仓库
scoop config scoop_repo 'https://gitcode.com/xrgzs/scoop'
scoop config scoop_branch 'master'
scoop update

# 设置 GitHub 加速镜像
scoop config GH_PROXY ghfast.top
```

配置完成后，每次 `scoop install` 都会自动检测地区并替换下载链接：

```
INFO  [UrlReplace] Detecting IP region (API: Cloudflare)...
INFO  [UrlReplace] You are in CN.
[UrlReplace] Result: https://ghfast.top/https://github.com/...
```

> [!tip]
> 如果 `scoop update` 超时，是因为 git 需要拉取完整的提交历史。可以强制更新：
> ```powershell
> Remove-Item -Path "~\scoop\apps\scoop\current\.git\" -Recurse -Force
> scoop update
> ```

### 维护技巧

```powershell
# 定期清空缓存，节省磁盘空间
scoop cache rm *

# 查看当前配置
scoop config

# 查看已安装的 bucket
scoop bucket list
```

> [!note]
> - 切换 scoop 源码仓库**不会影响已安装的软件**，因为软件位于 `~/scoop/apps/` 目录，与 scoop 源码独立。
> - 某些软件（如 TranslucentTB）需要管理员权限写入 Program Files，安装失败时可用 `sudo scoop install`，或手动执行安装器后再 `scoop shim add` 补全状态记录。

## 系统增强：PowerToys

Microsoft 官方出品的工具集，每一个小工具都直击痛点：

- **FancyZones** — 窗口布局管理。按 Shift 拖拽即可把窗口吸附到预设分区，比系统自带的二分屏强太多。适合大屏/带鱼屏用户。
- **PowerToys Run** — 快捷启动器。Alt+Space 唤出，比 Windows 搜索快得多，支持文件搜索、计算器、进程管理、插件扩展。
- **Keyboard Manager** — 按键重映射。比如把 CapsLock 映射为 Ctrl，或者把 F1-F12 映射为自定义快捷键。
- **Text Extractor** — 屏幕取词 OCR。按 Win+Shift+T 框选区域即可复制文字。
- **Peek** — 文件预览。选中文件按 Ctrl+Space 即可预览内容，类似 macOS 的 Quick Look。
- **File Locksmith** — 查看哪些进程占用了文件。
- **Always on Top** — 窗口置顶快捷键 Win+Ctrl+T。

所有工具都在 PowerToys Settings 里按需启用，不会吃资源。

## 文件搜索：Everything

Windows 自带的搜索慢得令人绝望。Everything 是文件搜索的终局方案：

- 基于 NTFS 索引，**实时更新**，输入即显示结果
- 支持正则、通配符、布尔表达式
- 体积不到 2MB，内存占用极低
- 可以通过 `efs://` 协议从浏览器调用
- **内置 HTTP 文件服务器**：在 Everything 中开启，局域网内其他设备通过浏览器即可下载文件，无需 U 盘或第三方工具

### 小技巧：开启 Everything HTTP 服务器，手机/平板也能下载文件

场景：你想把电脑上的一个文件传给旁边的手机，手边没有数据线，又不想登录微信/QQ。

Everything 自带了一个轻量级 HTTP 文件服务器：

1. 打开 Everything → **工具** → **选项** → **HTTP 服务器**
2. 勾选 **启用 HTTP 服务器**
3. （可选）设置用户名和密码，避免局域网内他人随意访问
4. 默认端口为 `80`，如被占用可改为其他端口如 `8080`

然后在手机/平板的浏览器中输入：

```
http://你的电脑IP:端口
```

比如 `http://192.168.1.100:8080`

你会看到 Everything 的网页界面，搜索文件即可直接下载。**不需要安装任何 App，不需要登录账号，浏览器就够了。**

> [!warning]
> 注意：开启 HTTP 服务器后局域网内所有人都能访问你的文件索引。建议设置密码，或仅在需要时临时开启。

### 进阶用法：Everything 命令行

Everything 支持命令行调用，配合 `everything -s` 也可以从终端直接搜索。结合 Scoop 安装后，随时在终端里搜索文件：

```powershell
# 搜索包含 "report" 的文件并直接定位到文件夹
es.exe report -open-path
```

配合 PowerToys Run，基本上告别了在文件夹里翻来找去。

## 截图与录屏

### Greenshot（截图）

免费开源，轻量级截图工具。

- 支持区域截图、窗口截图、全屏截图、滚动截图
- 截图后直接编辑：标注、箭头、马赛克、高亮
- 一键保存到文件/剪贴板/打印机，也能上传到 Imgur

### ScreenToGif（录屏/GIF）

名字已经说明一切了。免费开源。

- 支持录屏、摄像头录制、白板绘制
- 导出为 GIF、视频、或逐帧图片
- 内置编辑器可以裁剪帧、调色、加水印

偶尔写文档需要录个操作步骤，它就是最好的选择。

## PDF 阅读：SumatraPDF

轻量、启动快、功能纯粹。对比 Adobe Acrobat 和 Chrome 内置阅读器：

- 启动速度是 Adobe 的 10 倍以上
- 支持 PDF、EPUB、MOBI、CHM、DjVu、CBR/CBZ
- 支持高亮、标注
- 支持命令行参数，可从终端直接调用 `sumatrapdf.exe doc.pdf`

SumatraPDF 是我见过唯一一个**打开 PDF 不需要等 5 秒**的阅读器。

## 压缩工具：7-Zip

老牌经典，性能碾压 WinRAR：

- 压缩/解压速度在同类中几乎是最快的
- 支持格式：7z、ZIP、RAR、TAR、GZIP、XZ 等几十种
- 集成到右键菜单，无需打开主界面

界面可能略显朴素，但效率类工具不需要好看，只需要快。

## 开发环境配置

### 终端

- **Windows Terminal** — 微软官方的现代终端，支持多 Tab、分屏、GPU 加速、Unicode 和 emoji
- **Alacritty** — GPU 加速的极简终端，配置走 YAML，启动速度快到几乎感觉不到

### Shell

```powershell
# 装好 Scoop 后，安装 PowerShell 7
scoop install pwsh
```

然后在 PowerShell 配置文件 `$PROFILE` 里添加 Aliases：

```powershell
# 常用 aliases
function ll { ls -l @args }
function la { ls -Force @args }
function .. { Set-Location .. }
function dev { Set-Location ~/dev }
function g { git @args }
```

### 字体

等宽字体对开发体验的提升是立竿见影的：

- **JetBrains Mono** — 目前最流行的开发者字体，连字(Ligature)设计优雅
- **Maple Mono** — 国产开源字体，字形圆润清晰，自带连字和 Nerd Font 补丁，终端和编辑器体验极佳
- **Cascadia Code** — 微软出品，配合 Windows Terminal 食用更佳
- **Fira Code** — 经典连字字体

通过 Scoop 安装：

```powershell
scoop install JetBrains-Mono-NF Maple-Mono-NF Cascadia-Code Fira-Code
```

## 其他效率小贴士

### 快捷键习惯

几个值得花时间肌肉记忆的快捷键：

| 快捷键 | 功能 |
|--------|------|
| Win+V | 打开剪贴板历史（Windows 自带） |
| Win+Shift+S | 系统截图（Windows 自带） |
| Win+数字 | 打开/切换到任务栏第 N 个程序 |
| Alt+Tab | 切换窗口（按住 Alt 不放可用 Tab 选择） |
| Ctrl+Shift+Esc | 直接打开任务管理器 |

### 保持配置同步

有了 Scoop，你的软件清单就是活的。我维护一个 `setup.ps1` 放在 GitHub Gist 里：

```powershell
# setup.ps1
scoop install firefox vscode everything sumatrapdf
scoop install 7zip greenshot screentogif ffmpeg
scoop install alacritty jetbrains-mono-NF
```

新电脑上只需要跑：

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm https://gist.github.com/xxx/setup.ps1 | iex
```

配合 dotfiles 管理配置文件，真正做到**一条命令恢复开发环境**。

### 系统初始优化

```powershell
# 显示隐藏文件和文件扩展名
Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced -Name Hidden -Value 1
Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced -Name HideFileExt -Value 0

# 关闭 Xbox Game Bar
Get-AppxPackage *xboxapp* | Remove-AppxPackage
```
