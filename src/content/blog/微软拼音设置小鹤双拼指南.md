## 前言

微软拼音自带的双拼方案没有小鹤双拼，需要我们自行设置。下面给出几种方案：

## 方法一：CMD

将如下代码复制粘贴到 cmd 中运行，即可正确添加小鹤双拼。

```cmd
reg add HKCU\Software\Microsoft\InputMethod\Settings\CHS /v UserDefinedDoublePinyinScheme0 /t REG_SZ /d "小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt" /f
```

## 方法二：注册表

将以下内容保存为 `.reg` 文件，双击导入即可：

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\SOFTWARE\Microsoft\InputMethod\Settings\CHS]
"EnableExtraDomainType"=dword:00000001
"Enable Double Pinyin"=dword:00000001
"DoublePinyinScheme"=dword:0000000a
"UserDefinedDoublePinyinScheme0"="小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt"
```

## 总结

以上方法可以很方便地添加小鹤双拼方案，避免手动逐一设置键位。导入后重启输入法即可生效。
