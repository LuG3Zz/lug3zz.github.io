# 📝 语法支持
#   - 标准 Markdown（GFM）
#   - [[Wikilink]] — 自动转为站内链接
#   - ![[image.png]] — 嵌入图片
#   - > [!note] / > [!warning] / > [!tip] — 折叠 callout 块
#   - 代码块可指定语言，会自动添加复制按钮和行号
#
# 📏 建议
#   - 正文每行不超过 80 字符（非强制）
#   - 段落之间空一行
#   - 中英文之间加空格（如 "使用 Astro 框架"）
#   - 使用 `##` 和 `###` 作为章节标题（`#` 保留给文章标题）
#
# 🔗 站内链接
#   - 使用绝对路径：`[关于页面](/about)`
#   - 使用 Wikilink：`[[其他文章标题]]`
#
# 🖼️ 封面图
#   - 建议尺寸 1200×630px 左右
#   - 放在 public/images/ 下，引用为 /images/filename.jpg
#
# 🔒 密码保护
#   - 设置 password 后，列表页会隐藏该文章
#   - 访问 /blog/<slug> 会显示密码输入框
#   - 密码明文存储，用于简单访问控制
#
# 🚀 发布流程
#   1. 将 draft 改为 false 或删除 draft 字段
#   2. 运行 npm run build 确认无错误
#   3. git add -A && git commit -m "add: 新文章 slug"
#   4. git push origin main 自动部署
#
# 📋 常用命令
#   - npm run dev     — 本地预览
#   - npm run build   — 生产构建
#   - npm run preview — 预览构建结果
